# Abstraction

## Definition

Abstraction focuses on what an object represents rather than how it is implemented.

---

## Why is it important?

- Reduces complexity
- Improves maintainability
- Encourages separation of concerns
- Promotes reusable software

---

## SignalForge Example

Packet represents raw telemetry.

It does not know:

- where the data came from
- how it will be decoded

Its only responsibility is to represent raw telemetry data.
